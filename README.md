# 🐉 Dragon Curve Fractal

A real-time interactive dragon curve fractal generator built with vanilla HTML, CSS, and JavaScript.

## Features

- **Animated drawing** — watch the dragon curve grow segment by segment
- **6 color schemes** — Aurora, Fire, Ocean, Rainbow, Neon, Monochrome
- **Adjustable iterations** — from 1 to 20 (up to ~1M segments!)
- **Speed control** — from very slow to instant
- **Line width control** — thin hairlines to bold strokes
- **Auto-animate mode** — watch iterations cycle from 1 to 18 and back
- **Responsive design** — works on desktop and mobile
- **Glassmorphism UI** — frosted glass control panel

## The Dragon Curve

The dragon curve is a self-similar fractal that can be generated by repeatedly folding a strip of paper in half. It's also described by the L-system:

- **Axiom:** `FX`
- **Rules:** `X → X+YF+`, `Y → -FX-Y`

This implementation uses the equivalent iterative turn-sequence approach where each iteration doubles the number of segments.

## Running Locally

Just open `index.html` in any modern browser. No build tools or dependencies needed!

```bash
# Or serve locally with any static server
npx serve .
# or
python3 -m http.server 8080
```

## Controls

| Control | Description |
|---------|-------------|
| Iterations | Number of fractal iterations (1-20) |
| Draw Speed | How fast the curve animates |
| Color Scheme | Choose from 6 palettes |
| Line Width | Stroke thickness |
| Draw | Start/pause the animated drawing |
| Reset | Clear the canvas |
| Animate | Auto-cycle through iterations |
